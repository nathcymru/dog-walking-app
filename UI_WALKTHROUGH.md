## UI Patterns Used

### Colors:
- **Primary**: Blue for buttons and interactive elements
- **Success**: Green for "active" status
- **Warning**: Yellow/Orange for "suspended" status
- **Medium/Gray**: For "left" status and secondary text
- **Danger**: Red for error messages

### Typography:
- **Headers**: Bold, larger size
- **Labels**: Medium weight, smaller size
- **Body Text**: Regular weight
- **Error Text**: Danger color, smaller size

### Spacing:
- Consistent padding using Ionic's ion-padding class
- Card-based layout for better organization
- Gap between elements for breathing room

### Icons:
- Ionicons used throughout for consistency
- Icons paired with labels for clarity
- Standard icons (phone, mail, person, location, etc.)

### Responsive Design:
- Grid layout adjusts based on screen size
- Forms stack vertically on mobile
- Cards resize for tablet/desktop views
- Touch-friendly tap targets
